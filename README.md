# Infor_UDA
Replication of INFORMATION-THEORETIC ANALYSIS OF UNSUPERVISED DOMAIN ADAPTATION, ICLR 2023. https://arxiv.org/pdf/2210.00706

Based on repository https://github.com/atuannguyen/KL ,patched a severe bug in the original repository to run KL on RotatedMNIST correctly.

Engineered the KL-GP. Exact numerical reproduction was not achieved, but the core theoretical claim is empirically validated by the relative performance gain.
